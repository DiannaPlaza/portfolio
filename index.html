Projects

YBS Galactic Tours

Html5 | Css3 | Javascript | React | Rest API

YBS Galactic Tours is a fake space tour company with an online preview for their upcoming line of space tours. The user can select an option from a number of space tours and be shown a view images of the location they will be visiting. YBS has their user's safety always in mind, so they only display upcoming tour dates that are free from asteroids! This application was created in a team programing environment using two of Nasa's REST APIs.

Quote Generator

Html5 | Css3 | React | API

Everyone needs a motivational quote at some point in their life. This solo project was created just for that while using React concepts and working with a third-party quote data source to display inspirational quotes to the user based on their category selection. I am currently working on adding Firebase to this project to allow users to save their favourite quotes to their Quote Journal.

What are we drinking tonight?

Html5 | Sass | Javascript | Rest Api

This application is a fun pair programing project that was created using Javascript and a Cocktail API. The user "picks their posion" of a base liqour which then displays images of all the drinks they can create using said liqour. The user can click on any of the drink options to display their ingredients and instructions. A perfect application to use after a long days work!

It's all about Sophie

Html5 | Css3 | Sass

Sophie is a multi-page, functional, and responsive website created from a provided PSD. This site is about a young blogger who shares her life experiences with her followers.
